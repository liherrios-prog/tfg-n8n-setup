# TFG n8n Setup

Este repositorio contiene todo lo necesario para desplegar rápidamente n8n en un entorno Docker, optimizado para entornos educativos.

## Características

- ✅ Instalación con un solo comando
- ✅ Base de datos SQLite persistente
- ✅ Backup y restore automatizados
- ✅ Configuración HTTPS opcional
- ✅ Integración con GitHub (opcional)
- ✅ Scripts de gestión completos

## Requisitos Previos

- Docker (v20.10+)
- Docker Compose (v2.0+)
- Git

## Instalación Rápida

```bash
git clone https://github.com/tu-usuario/tfg-n8n-setup.git
cd tfg-n8n-setup
./install.sh
